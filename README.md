# Basic Node.js CRUD API

This project demonstrates how to:

- Connect Node.js to MongoDB Atlas
- Use Mongoose for schema modeling
- Implement CRUD operations
- Structure backend using MVC
- Test APIs using Postman

## Tech Stack
- Node.js
- Express
- MongoDB Atlas
- Mongoose

## Run Project

1. Clone repository
2. Run `npm install`
3. Create `.env` file
4. Run `npm start`
